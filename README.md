# Primera Compañía de Bomberos de Temuco - Web 2025

[![Estado del Proyecto](https://img.shields.io/badge/Estado-En%20Desarrollo-yellow.svg)](https://github.com/SausageFrog/bomberos-2025/actions)

## Resumen

Desarrollo de una pagina responsiva y optimizada para la Primera Compañía de Bomberos de Temuco. El objetivo central es la integración de servicios digitales para fortalecer la conexión comunitaria, la transparencia operativa y la sostenibilidad financiera de la institución mediante herramientas web modernas.

## Funcionalidades Implementadas

La plataforma aborda tres áreas funcionales críticas para la modernización institucional:

1.  **Módulo de Audio en Vivo (Radio):** Integración de un reproductor embebido de baja latencia, permitiendo a la comunidad escuchar la frecuencia de despacho en tiempo real.
2.  **Sistema de Donaciones (Crowdfunding):** Interfaz segura para la captación de recursos, ofreciendo modalidades de **suscripción mensual** y **donaciones únicas**, integradas mediante estándares PCI DSS.
3.  **Directorio Interactivo del Cuerpo:** Visualización tipo grilla de todas las compañías de Temuco, presentando identidad visual (logos) y **enrutamiento directo** a sus sitios web oficiales para fomentar la colaboración.

##  Instalación y Ejecución Local

Dado que el proyecto utiliza tecnologías **web nativas** (HTML, CSS, JavaScript), no se requieren dependencias externas ni compiladores (`npm`).

Para ejecutar la aplicación localmente:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/SausageFrog/bomberos-2025.git](https://github.com/SausageFrog/bomberos-2025.git)
    ```
2.  **Acceder a la carpeta del proyecto:**
    ```bash
    cd bomberos-2025
    ```
3.  **Abrir la aplicación:**
    Localiza el archivo `index.html` en la carpeta raíz y **ábrelo directamente en tu navegador web** (Chrome, Firefox, Edge, etc.).
